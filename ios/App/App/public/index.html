How to Unblock GPS

To use the GPS rangefinder, you must allow location permissions. Follow these steps:

📱 iPhone Home Screen Bookmarks

  1. Open your iPhone's main Settings app.
  2. Go to Privacy & Security > Location Services.
  3. Find "GolfCaddie AI" (or the bookmark's title) or your browser (like Safari Websites or Chrome) in the list.
  4. Ensure location access is set to While Using the App or Ask Next Time.
  5. Force-close the Home Screen app (swipe it up/away), then re-open it.
  6. Tap the red GPS Offline badge to trigger the permission prompt and sync location!

⚠️ Stuck Permissions? If iOS refuses to show the location popup after tapping the badge, go to your iPhone's Settings > General > Transfer or Reset iPhone > Reset and tap Reset Location & Privacy, then reopen the app.

📱 iPhone (Safari Browser)

  1. Tap the aA icon in the address bar search box.
  2. Select Website Settings.
  3. Set Location to Allow.
  4. Refresh the page!

📱 iPhone (Chrome Browser)

  1. Open your iPhone's main Settings app.
  2. Scroll down through your apps list and tap Chrome.
  3. Tap Location and select While Using the App.
  4. Return to Chrome and refresh the page!

💻 Desktop (Chrome)

  1. Click the settings/padlock icon to the left of the URL.
  2. Set Location to Allow.
  3. Reload the page!
